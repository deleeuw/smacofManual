Directional Derivatives

Let \(\sigma_k(X):=w_k|d_k(X)-\delta_k|\).

  1. If \(\delta_k=0\) and \(d_k(X)=0\) then \(d\sigma_k(X;Y)=w_kd_k(Y)\).
  2. If \(\delta_k>0\) and \(d_k(X)=0\) then \(d\sigma_k(X;Y)=-w_kd_k(Y)\).
  3. If \(d_k(X)>0\) and \(d_k(X)-\delta_k>0\) then \(d\sigma_k(X;Y)=w_k\frac{1}{d_k(X)}\text{tr}\ X'A_kY\).
  4. If \(d_k(X)>0\) and \(d_k(X)-\delta_k<0\) then \(d\sigma_k(X;Y)=-w_k\frac{1}{d_k(X)}\text{tr}\ X'A_kY\).
  5. If \(d_k(X)>0\) and \(d_k(X)-\delta_k=0\) then \(d\sigma_k(X;Y)=w_k\frac{1}{d_k(X)}|\text{tr}\ X'A_kY|\).

Smoothing

Smoothing: Pseudo-Huber Loss

\[ \sigma_\epsilon(X):=\sum w_k\sqrt{(\delta_k-d_k(X))^2+\epsilon^2} \] Now majorization using \[ \sqrt{(\delta_k-d_k(X))^2+\epsilon^2}\leq\frac12\frac{1}{\sqrt{(\delta_k-d_k(Y))^2+\epsilon^2}}\{(\delta_k-d_k(X))^2+(\delta_k-d_k(Y))^2+2\epsilon^2\} \] Alt:

\[ \sigma_\epsilon(X)=\epsilon^2\left\{\sqrt{1+\frac{x^2}{\epsilon^2}}-1\right\} \]

Robustifying - Huber Loss

The Huber function is \[ f(x)=\begin{cases} \frac12x^2&\text{ if }|x|<c,\\ c|x|-\frac12 c^2&\text{ otherwise}. \end{cases} \]

The Huber function is differentiable, although not twice diffentiable. Its derivative is \[ f'(x)=\begin{cases} c&\text{ if }x\geq c,\\ x&\text{ if }|x|\leq c,\\ -c&\text{ if }x\leq -c. \end{cases} \] We now use the results of De Leeuw and Lange (2009), specifically Theorem 4.5 and example 5.3. For each \(y\) we define a quadratic in \(x\) by \[ g(x,y):=\frac{f'(y)}{2y}(x^2-y^2)+f(y) \] The theorem says that if

  • \(f\) is even,
  • \(f\) differentiable, and
  • \(f'(x)/x\) is decreasing on \((0,+\infty)\)

then \(g(\bullet,y\) majorizes \(f\), i.e.

  • \(f(x)\leq g(x, y)\) for all \(x,y\), and
  • \(f(x)=g(x,x)\) for all \(x\).

Moreover if \(h\) is another quadratic majorizing \(f\) at \(y\) then \(h(x)\geq g(x)\) for all \(x\). Thus \(g\) is a sharp quadratic majorizer.

The Huber function is even and differentiable. Moreover \(f'(x)/x\) decreases from. Thus the theorem applies and the sharp quadratic majorizer at \(y\) is \[ g(x)=\begin{cases} \end{cases} \]

\[ \sigma_k(X)=\begin{cases} \frac12(\delta_k-d_k(X))^2&\text{ if }|\delta_k-d_k(X)|<c,\\ c|\delta_k-d_k(X)|-\frac12 c^2&\text{ if }|\delta_k-d_k(X)|\geq c. \end{cases} \]

\[ \omega_k(x,y)=\begin{cases} \frac12\frac{c}{|y|}(x^2-y^2)-cy-\frac12c^2&\text{ if }y\leq -c,\\ \frac12x^2&\text{ if }|y|<c,\\ \frac12\frac{c}{|y|}(x^2-y^2)+cy-\frac12c^2&\text{ if }y\geq +c. \end{cases} \] Now \(x=\delta_k-d_k(X)\) and \(y=\delta_k-d_k(Y)\)

\[ \omega_k(X;Y)=\begin{cases} \frac12\frac{c}{|\delta_k-d_k(Y)|}\{(\delta_k-d_k(X))^2+(d_k(Y)-\delta_k)^2\}-c(\delta_k-d_k(Y))-\frac12c^2&\text{ if }\delta_k-d_k(Y)\leq -c,\\ \frac12(\delta_k-d_k(X))^2&\text{ if }|\delta_k-d_k(Y)|<c,\\ \frac12\frac{c}{|\delta_k-d_k(Y)|}\{(\delta_k-d_k(X))^2+(d_k(Y)-\delta_k)^2\}+c(\delta_k-d_k(Y))-\frac12c^2&\text{ if }\delta_k-d_k(Y)\geq +c. \end{cases} \] Thus the MDS majorization algorithm for the Huber loss is to update \(Y\) by minimizing (or by performing one smacof step to decrease) \[ \sum w_kv_k(Y)(\delta_k-d_k(X))^2 \] where \[ v_k(Y)=\begin{cases} 1&\text{ if }|\delta_k-d_k(Y)|<c,\\ \frac{c}{|\delta_k-d_k(Y)|}&\text{ otherwise}. \end{cases} \]
